# Project Charter: ChemiCrown CDMS

## 1. Project Information
- **Project Name:** ChemiCrown Distribution Management System (CDMS)
- **Client:** ChemiCrown (Bhavnagar, Gujarat)
- **Duration:** 6 Weeks
- **Team Size:** 3 Interns

## 2. Executive Summary
ChemiCrown is a wholesaler and distributor of industrial chemicals. The CDMS will digitize their operations, moving them from manual tracking to a secure, role-based web platform. The system will handle public product inquiries, internal inventory management, supplier tracking, and customer orders.

## 3. Project Objectives
- Create a public-facing catalog for products (GP Thinner, MTO, Toluene, Acetone, etc.).
- Implement a Customer Portal for order tracking and quotation requests.
- Build an Internal Dashboard for Managers to track Inventory KPIs and Orders.
- Ensure strict Role-Based Access Control (RBAC).

## 4. Scope
### In-Scope (MVP)
- Public Website (Home, About, Products, Contact).
- User Authentication (Supabase Auth).
- Inventory Management (CRUD, Transactions).
- Order Management Pipeline.
- Audit Logging & Soft Deletes.

### Out-of-Scope (Future Enhancements)
- Integrated Payment Gateway (Payments handled offline for MVP).
- Advanced Logistics Tracking.

## 5. Roles & Responsibilities
- **Intern 1 (Project Lead):** Architecture, DB Schema, Auth, DevOps.
- **Intern 2:** Public Website, UI/UX, Product Pages.
- **Intern 3:** Internal Dashboard, Order Flows, Reports.
